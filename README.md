# BuildSpace – Developer Collaboration Platform

A responsive web app prototype that unifies:
- Developer profiles
- Project/team formation
- Opportunity posting/discovery
- Interactive community feed

## Run locally

```bash
python3 -m http.server 8080
```

Then open `http://localhost:8080`.

## Included features

- Developer profile create/update with skills/interests/projects
- Project creation with description, stack, and collaboration need
- Opportunity board for teammate/hiring/hackathon posts
- Interactive feed combining profile updates, projects, and opportunities
- Search/filter across feed content
- Dark/light theme toggle
- Local storage persistence
- Responsive layout for mobile + desktop
